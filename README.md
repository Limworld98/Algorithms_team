# 로봇청소기의 경로 탐색 알고리즘
<h3>컴퓨터알고리즘과 실습 팀프로젝트</h3> 
벽과 장애물이 존재하는 실내에서 전체 공간에 대한 정보가 없다고 가정할 경우, 로봇청소기가 효율적으로 청소할 수 있는 경로 탐색 알고리즘을 구현한다.
배터리를 고려한 시간 제한이 존재하는 방식과, 시간 제한을 두지 않은 두가지 방식으로 구현하였다.<br><br>
